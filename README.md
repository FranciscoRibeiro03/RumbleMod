RumbleRumbleQuest
=================

![cover](./cover.gif)

Quest BeatSaber mod for altering haptic feedback behaviors. A port of the RumbleMod to the Quest.

## Settings

The preferred way is to use the in-game UI for configuration.

![config](./config.jpg)
